# Sample Images

This folder contains sample potato leaf images that you can use to test the application.

| File | Disease Type |
|------|--------------|
| early_blight_sample.jpg | Early Blight |
| late_blight_sample.jpg | Late Blight |
| healthy_sample.jpg | Healthy |

## How to Test

1. Start the application (both backend and frontend)
2. Open http://localhost:3000 in your browser
3. Upload any of these images
4. See the prediction result
